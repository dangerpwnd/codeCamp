What is a flexbox

  • This intro is a glimpse at the css layout element, flexbox.

  • Since it's first working draft in 2009, it's developed into W3C Candidate Recommendation and is still being actively updated.

It can do what

  • Using flex is as easy as adding this code to the parent container:

  • display:flex;

  • This new parent container will hold all of your flex-items.

  • The flexbox container has two axes: a main axis and a cross axis.

  • flex-direction:column

  • The flex-direction property that will arrange the items along the main axis. By default, these items will be displayed left to right.

Some Properties

  • A few additional properties to dive in to include the following:

  • flex-wrap: wrap

  • By default all flex items will be one line, with flex-wrap you can allow items to wrap as needed.

  • flex-flow: column wrap

  • A shorthand for flex-direction and flex-wrap.

  • justify-content:center

  • Defines alignment along the main axis. Flex-start(similar to float:left) is the default.

  • align-items:center

  • Defines aligment along the cross axis. Stretch(fill) is the default.

Children everywhere

  • These child elements are going to be shaped partial by the parent, but are also going to be their own special little snow flake.

  • These child properties can customize individual child elements:

  • flex-grow: 2

  • Defines how much space to take up in flex container.

  • flex-shrink: 2

  • >Defines how much flex item should shrink if necessary.>Defines how much flex item should shrink if necessary.

  • flex-basis: 2

  • Defines default size of an element before the remaining space is distributed.

  • flex: 5

  • A shorthand for flex-grow, flex-shrink, and flex-basis.

Plays well with others

  • While some advocate Flexbox over Grid, or vise-versa, the truth is they work well together. A good implementation would be as follows:

  • .main-container { display: grid; grid-template-rows: repeat(2, 1fr); } .child-container { display: flex; justify-content: center;}
